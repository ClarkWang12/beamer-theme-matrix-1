Another Beamer Theme Matrix

As you probably know, Beamer is a LaTeX package for creating presentation slides. This matrix shows examples of all variations of the default themes and color themes.

The matrix’s rows list the default Beamer themes, and the matrix’s columns list the default Beamer color themes. In a LaTeX document’s preamble, set the theme using \usetheme{Berlin} and set the the color theme using \usecolortheme{crane}, where Berlin and crane are replaced with the theme and color theme of your choosing.

The code used to generate this page is available on GitHub. This page was inspired by the original Beamer Theme Matrix.
